# Here-sMyPet
A spring board to show off your Pet

1. 프로젝트 소개
- 각자의 반려동물 사진을 올려서 자랑하고, 선플로 응원하는 게시판입니다.

2. 프로젝트 기능
프로젝트의 주요 기능은 다음과 같습니다.

게시판 - CRUD 기능, 조회수, 페이징
사용자 - 관리자/로그인/회원가입으로 권한 설정
댓글 - CRUD 기능

3. 팀소개 (백엔드 4명)
- 게시글 관리팀: 원준표, 권진우
- 회원 관리팀: 진성길, 백서준

4. 사용 기술
- 주요 프레임워크: Java, Flask, Python
- DB: MySQL
- 프런트엔드: HTML, CSS
 
5. 실행 화면
